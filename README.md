# Python Task Automation & Scheduler

A lightweight Python tool for scheduled email, web scraping, and file organization.

## Features
- Automated email sending (SMTP)
- Web scraping and CSV/Excel export
- File organization by extension
- Task scheduling (hourly, daily, weekly)
- Logging to `log.txt`

## Setup
1. Python 3.8 required
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
3. Configure `config.json` with your settings
4. Run:
   ```sh
   python main.py
   ```

## Deployment
- Run manually or schedule with Windows Task Scheduler/Cron
